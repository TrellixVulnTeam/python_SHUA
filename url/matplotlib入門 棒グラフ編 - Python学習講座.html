簡単なサンプル

まずは最低限、簡単なサンプルからです。

1
2
3

import matplotlib.pyplot as plt
 
left = [1, 2, 3, 4, 5] # 横軸(棒の左端の位置)
height = [3, 5, 1, 2, 3] # 値
plt.bar(left, height)
plt.show()

実行結果


leftで横軸、heightで値を設定します。leftはあくまでも横軸の座標、つまり棒の左端の位置である、ということを抑えておくと複雑なグラフを描く際役に立つと思います。

例えば、上記リストをleft = [-1, 2.5, 3, 4, 7]と指定した場合、以下のようになります。マイナスや、小数の指定が可能です。幅のデフォルトは0.8なので、２番めの３番目の棒に重なりができています。
left = [-1, 2.5, 3, 4, 7]の場合

オプション

最低限の表示ができたところで、オプションについて確認してみましょう。bar関数の引数は以下の通りです。

引数 必須 意味 指定する値
left ○ 横座標 数値シーケンス
height ○ 縦座標 数値シーケンス
width 棒の幅 数値。デフォルトは0.8
bottom 棒の下側の位置 #RGB
color 房の色 #RGB
edgecolor 棒の枠線の色 数値
linewidth 棒の枠線の太さ 数値
tick_label 横軸のラベル 文字列シーケンス
xerr 横軸のエラーバー 数値シーケンス
yerr 縦軸のエラーバー 数値シーケンス
ecolor エラーバーの色 #RGB
align 棒の位置 ‘edge’ 、‘center’ のどちらかを指定。デフォルトは’edge’
log 対数目盛り TrueかFalseのどちらかを指定。デフォルトはFalse

それでは、上記オプションを踏まえて色んな棒グラフを作成してみましょう。

棒の間隔を詰めてみる

ヒストグラム等、棒と棒の間隔を開けたくない場合どうすればよいでしょうか？leftの間隔と棒の幅を同じにすると、間隔がなくなります。（ただし、一般的にヒストグラムはhist関数を利用します。）

import numpy as np
import matplotlib.pyplot as plt
 
left = [1, 2, 3, 4, 5]
height = [3, 5, 1, 2, 3]
labels = ['A', 'B', 'C', 'D', 'E']
plt.bar(left, height, tick_label=labels, align='center', width=1)
plt.show()

最大/最小を指定する(グラフの天井との間隔を空ける)

棒グラフの機能ではないのですが、デフォルトだと棒の最大値とグラフの天井が一致してしまいます。体裁を整えたい場合、plt.ylimを指定します。

import matplotlib.pyplot as plt
 
left = [1, 2, 3, 4, 5]
height = [3, 5, 1, 2, 3]
labels = ['A', 'B', 'C', 'D', 'E']
plt.bar(left, height, tick_label=labels, align='center')
 
# スケールの最大/最小を指定する
plt.ylim(ymax = 25, ymin = 0)
 
plt.show()

複数の系列を描画する

複数の系列を描画する場合はちょっと工夫が必要になります。２系列目以降はleftに棒の幅分ずらした値を指定する必要があります。デフォルトの幅が0.8なので、その半分の0.4を指定しています。また、横軸のラベル位置は、bar関数で指定すると、棒の真下にしか描けません。このため、xtics関数で別途指定する必要がでてきます。

# ※後述しますが、良くないサンプルです。
import matplotlib.pyplot as plt
 
w = 0.4 # 棒の幅
left1 = [1, 2, 3, 4, 5]
left2 = [1 + w, 2 + w, 3 + w, 4 + w, 5 + w] # 棒の幅分ずらす
 
height1 = [3, 5, 1, 2, 3]
height2 = [6, 10, 2, 4, 6]
labels = ['A', 'B', 'C', 'D', 'E']
plt.bar(left1, height1, align='center', color='#FFA0A0', width=w, label='系列1')
plt.bar(left2, height2, align='center', color='#A0A0FF', width=w, label='系列2')
plt.legend() # 凡例を出力する
 
# 横軸のラベルはxticksで別途指定する
label_left = [1 + w/2, 2 + w/2, 3 + w/2, 4 + w/2, 5 + w/2]
plt.xticks(label_left, labels)
plt.show()

import numpy as np
import matplotlib.pyplot as plt
 
height1 = [3, 5, 1, 2, 3]
height2 = [6, 10, 2, 4, 6]
 
# 横軸の設定
w = 0.4 # 棒の幅
left = np.arange(len(height1))
 
labels = ['A', 'B', 'C', 'D', 'E']
plt.bar(left, height1, align='center', color='#FFA0A0', width=w, label='系列1')
plt.bar(left + w, height2, align='center', color='#A0A0FF', width=w, label='系列2')
plt.legend() # 凡例を出力する
 
# 横軸のラベルはxticksで別途指定する
plt.xticks(left + w/2, labels)
plt.show()

複数の系列を描画する(積み上げ棒グラフ)

積み上げ棒グラフで複数の系列を表示する場合は簡単で、２番目以降の系列に対し、bottomオプションで１番目の系列の値分、底上げすればOKです。

import matplotlib.pyplot as plt
 
left = [1, 2, 3, 4, 5]
height1 = [3, 5, 1, 2, 3]
height2 = [6, 10, 2, 4, 6]
 
labels = ['A', 'B', 'C', 'D', 'E']
plt.bar(left, height1, align='center', color='#FFA0A0', label='系列1')
plt.bar(left, height2, align='center', color='#A0A0FF', label='系列2', bottom=height1) # 系列1の高さ分、bottomを指定して底上げする
plt.legend() # 凡例を出力する
plt.xticks(left, labels) # 横軸ラベル
 
plt.show()
