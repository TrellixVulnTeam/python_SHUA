辞書のサイズの取得(len関数)

広告

辞書オブジェクトが現在持っている要素数(キーの数)を取得することが出来ます。組み込み関数の「len」関数を使用します。 

len(リストオブジェクト)

辞書に含まれる要素数を取得します。「len」関数は辞書の他にシーケンス型のリストや文字列、タプルなどの要素数を取得できます。

具体的には次のように記述します。 

dict = {"yamada":75, "endou":82}

print len(list)  # 2

上記の場合はリストの要素数「2」を表示します。

サンプルコード

では実際に試してみます。

  • test4-1.py
#coding: UTF-8

dict = {"yamada":75, "endou":82}
print dict
print u"要素数=" + str(len(dict))

上記を「test4-1.py」として保存します。保存する時の文字コードはUTF-8です。そして次のように実行して下さい。
